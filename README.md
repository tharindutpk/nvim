# Tharindu's Neovim config

Here's my personal Neovim configuration, meticulously crafted for `Golang`, `Rust`, `JavaScript/TypeScript`, and `Python` development, with occasional `C/C++` support. Pair it with `tmux` for exceptional productivity.

| Language  | LSP                | Formatter            | Linter               |
| ---       | ---                | ---                  | ---                  |
| Go        | gopls              | goimports & gopls    | -                    |
| Rust      | rust-analyzer      | rustfmt              | -                    |
| Python    | pyright            | ruff                 | ruff                 |
| JS/TS     | tsserver           | eslint_d             | eslint_d             |
